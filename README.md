# Модуль 15-1. Подключение к SQL Server из Python

## Описание проекта
Проект демонстрирует подключение к Microsoft SQL Server из Python с использованием библиотеки pyodbc. Скрипты создают базу данных "FruitsAndVegetables", таблицу с продуктами, заполняют её 10 записями и выводят данные в виде списка словарей.

## Состав проекта

### База данных FruitsAndVegetables
Таблица продуктов (овощи и фрукты):
- Id - первичный ключ (автоинкремент)
- Name - название продукта (уникальное)
- Type - тип (овощ/фрукт)
- Color - цвет
- Calories - калорийность
- Description - краткое описание

### Файлы проекта
- simple_queries.py - основной скрипт с логикой работы
- SQLQueries.py - модуль с SQL-запросами
- queries.sql - SQL скрипт создания таблицы
- requirements.txt - зависимости проекта
- .env_sample - шаблон файла конфигурации
- .gitignore - исключенные из git файлы

## Выполняемые операции
Скрипт последовательно выполняет:
1. Создание базы данных FruitsAndVegetables
2. Создание таблицы с ограничениями (PRIMARY KEY, UNIQUE)
3. Заполнение таблицы 10 записями (фрукты и овощи)
4. Вывод данных в табличном формате и в виде списка словарей

## Требования
- Python 3.x
- Microsoft SQL Server (любая редакция)
- Библиотеки: pyodbc, python-dotenv, requests

## Установка и запуск
1. Клонировать репозиторий
2. Создать виртуальное окружение: python -m venv .venv
3. Активировать: .venv\Scripts\activate (Windows)
4. Установить зависимости: pip install -r requirements.txt
5. Создать файл .env по образцу .env_sample с вашими данными подключения
6. Запустить: python simple_queries.py
